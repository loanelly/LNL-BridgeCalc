## 📖 Обзор

**LNL-BridgeCalc** — это ориентированное на пользователя десктопное программное обеспечение, разработанное для автоматизации проверок целостности строительных конструкций, расчета предельных состояний и проецирования активных векторов сил в режиме реального времени. Созданное специально для защиты инфраструктуры и предотвращения катастрофических обрушений мостов, это приложение заменяет слепые экспертные оценки строгими физическими уравнениями.

## 🚀 Ключевые возможности

*   **Ядро расчета 5 топологий мостов**: Специализированные уравнения деформации и тригонометрические формулы для **Балочных**, **Арочных**, **Вантовых**, **Висячих** и **Рамных** мостов.
*   **Динамическая перестройка интерфейса**: Графический интерфейс адаптируется на лету, генерируя уникальные поля ввода параметров под инженерные особенности выбранного типа моста.
*   **Интерактивный физический холст Canvas**: Динамически отрисовывает масштабируемые чертежи мостов, накладывая активные траектории векторов нагрузок ($H_{thrust}$, $T_{max}$, $R_1, R_2$, $N_{col}$) в реальном времени.
*   **Оценка коэффициента запаса прочности ($K_{safe}$)**: Мгновенно рассчитывает запасы прочности элементов, предупреждая пользователя явным инженерным заключением в случае угрозы разрушения.
*   **Автоматический экспорт на диск**: Генерирует структурированные и детальные файлы отчетов (`.txt`) в локальной директории при каждом цикле вычислений.

## 🏗️ Техническая архитектура и профили параметров

| Тип моста | Ключевые входные параметры | Контролируемые силы и метрики | Векторные маркеры на чертеже |
| :--- | :--- | :--- | :--- |
| **Балочный мост** | Ширина ($b$), Высота ($h$), Полезная нагрузка ($q_{live}$), $k_{dyn}$ | Напряжение изгиба ($\sigma_{act}$), Макс. прогиб ($f_{max}$) | $P_{live} + G_{self}$, Реакции опор $R_1, R_2$ |
| **Арочный мост** | Подъем свода ($H$), Толщина арки ($t$), Полотно $b \times h$ | Распор фундамента ($H_{thrust}$), Сжатие свода ($N_{max}$) | Центр масс $G$, Распор $H_{thrust}$, Реакция устоя $V$ |
| **Вантовый мост** | Высота пилона ($H$), Кол-во вант, Полотно $b \times h$ | Натяжение вант ($T_{max}$), Сжатие пилона ($P_{pylon}$) | Сила в ванте $T_{max}$, Вертикаль пилона $P_{pylon}$ |
| **Висячий мост** | Провес кабеля ($f$), Диаметр троса ($d$), Полотно $b \times h$| Натяжение у пилонов ($T_{support}$), Тяга анкера ($H_{anchor}$) | Натяжение нити $T_{support}$, Удержание анкера $H_{anchor}$ |
| **Рамный мост** | Толщина стоек ($t$), Высота стоек ($H$), Полотно $b \times h$| Жесткий опорный момент ($M_{support}$), Осевое напряжение ($\sigma_{comb}$) | Сжатие стойки $N_{col}$, Эпюра моментов узла $M_{joint}$ |

## 📊 Стандарты безопасности ($K_{safe}$)

Программный комплекс оценивает механические напряжения материалов относительно временного сопротивления разрушению ($\sigma_B$) по методу расчета предельных состояний:

$$\text{Коэффициент запаса } (K_{safe}) = \frac{\text{Предел прочности } (\sigma_B)}{\text{Действующее напряжение } (\sigma_{act})}$$

*   🟢 **$K_{safe} \ge 1.5$** &rarr; `APPROVED`: Спецификация одобрена. Конструкция полностью стабильна под расчетными нагрузками.
*   🟡 **$1.0 \le K_{safe} < 1.5$** &rarr; `CRITICAL HAZARD`: Опасность! Инициирована пластическая деформация. Надежность под угрозой.
*   🔴 **$K_{safe} < 1.0$** &rarr; `COLLAPSE FAILURE`: Разрушение системы. Немедленный излом материалов. Мост рухнет.

## 📦 Инструкция по сборке и запуску

### Требования к окружению
*   Java JDK 17 или выше
*   Apache Maven 3.8+

### Компиляция автономного Fat JAR
Запустите цикл сборки Maven для автоматической упаковки кроссплатформенных графических модулей JavaFX внутрь одного исполняемого файла:
```bash
mvn clean package
```

### Запуск приложения
Запустите сгенерированный бинарный файл из папки target или перенесите его в удобное место для запуска двойным кликом:
```bash
java -jar target/LNL-BridgeCalc-1.0-SNAPSHOT.jar
```
