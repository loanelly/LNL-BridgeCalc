1. ОБЗОР СИСТЕМЫ И АРХИТЕКТУРА
-------------------------------------------------------------------------------
LNL-BridgeCalc — это передовой инженерный комплекс автоматизированного 
проектирования, прочностного контроля и мониторинга безопасности искусственных
сооружений.

Архитектура системы разделена на изолированные уровни логики:
* Физическое ядро (BridgesDatabase.java): Расчет уравнений статики и прочности.
* Уровень управления (HelloController.java): Управление жизненным циклом JavaFX.
* Векторный графический движок (Canvas API): Масштабирование чертежных узлов.

2. ТЕОРЕТИЧЕСКАЯ МЕХАНИКА И СПЕЦИФИКАЦИИ ТОПОЛОГИЙ
-------------------------------------------------------------------------------

[ТИП 1: БАЛОЧНЫЙ МОСТ]
* Конструктивная схема: Горизонтальные пролеты, свободно лежащие на опорах.
* Характер деформации: Чистый изгиб полотна и поперечный сдвиг устоев.
* Входные инженерные параметры:
  - b (Ширина полотна настила, метры)
  - h (Высота сечения несущей балки, метры)
  - liveLoad (Временная полезная нагрузка от автотранспорта, кН/м)
  - kDyn (Коэффициент динамичности/удара колес)
* Математическая модель:
  - Площадь сечения балки: A = b * h
  - Собственный погонный вес: q_self = A * density * G
  - Полная расчетная погонная нагрузка: q_total = q_self + (liveLoad * 1000 * kDyn)
  - Максимальный изгибающий момент: M_max = (q_total * L^2) / 8
  - Момент сопротивления прямоугольного бруса: W = (b * h^2) / 6
  - Действующее нормальное напряжение: sigma_act = M_max / W
  - Стрела прогиба балки: f_max = (5 * q_total * L^4) / (384 * E * I)

[ТИП 2: АРОЧНЫЙ МОСТ]
* Конструктивная схема: Распорная система с криволинейным несущим сводом.
* Характер деформации: Чистое осевое сжатие дуги и передача бокового распора.
* Входные инженерные параметры:
  - b (Ширина полотна настила, метры)
  - h (Высота сечения полотна настила, метры)
  - H_arch (Стрела подъема купола арки, метры)
  - t_arch (Толщина поперечного сечения дуги арки, метры)
* Математическая модель:
  - Угол заложения оси арки у опор: alpha = arctan(H_arch / (L / 2))
  - Полный вес конструкции: w_total = Вес настила + Вес свода арки
  - Горизонтальный распор фундамента: H_thrust = (w_total * L) / (8 * H_arch)
  - Сжимающее продольное усилие на опоре: N_max = H_thrust / cos(alpha)
  - Напряжение сжатия в сечении арки: sigma_comp = N_max / (t_arch * b)

[ТИП 3: ВАНТОВЫЙ МОСТ]
* Конструктивная схема: Полотно удерживается наклонными канатами из пилона.
* Характер деформации: Растяжение в вантах, вертикальное сжатие в пилоне.
* Входные инженерные параметры:
  - b (Ширина полотна настила, метры)
  - h (Высота сечения полотна настила, метры)
  - H_pylon (Высота центрального пилона над полотном, метры)
  - count (Количество удерживающих косых вант в пучке, штук)
* Математическая модель:
  - Усредненный угол схода канатов: beta = arctan((L / 2) / H_pylon)
  - Пиковое растяжение крайней ванты: T_max = (W_deck / 2) / (cos(beta) * count)
  - Вертикальное сжатие тела пилона: P_pylon = Полный вес дорожного настила
  - Напряжение растяжения стали вант: sigma_cable = T_max / Площадь кабеля

[ТИП 4: ВИСЯЧИЙ МОСТ]
* Конструктивная схема: Полотно подвешено к гибким кабелям параболической формы.
* Характер деформации: Чистое растяжение главного троса, анкерное удержание.
* Входные инженерные параметры:
  - b (Ширина полотна настила, метры)
  - h (Высота сечения полотна настила, метры)
  - f_cable (Стрела провеса центральной нити кабеля, метры)
  - d_cable (Внешний диаметр сечения несущего стального кабеля, метры)
* Математическая модель:
  - Площадь сечения троса: A_cable = pi * (d_cable / 2)^2
  - Горизонтальный распор гибкой нити: H_thrust = (q_deck * L^2) / (8 * f)
  - Максимальное натяжение у пилонов: T_support = H_thrust * sqrt(1 + 16*(f/L)^2)
  - Напряжение в металле кабеля: sigma_act = T_support / A_cable

[ТИП 5: РАМНЫЙ МОСТ]
* Конструктивная схема: Жесткое монолитное соединение ригеля со стойками.
* Характер деформации: Изгиб ригеля, внецентренное сжатие опорных колонн рамы.
* Входные инженерные параметры:
  - b (Ширина полотна настила, метры)
  - h_beam (Высота сечения горизонтального ригеля рамы, метры)
  - t_frame (Толщина сечения вертикальных опорных стоек рамы, метры)
  - H_frame (Полная вертикальная высота опорных стоек рамы, метры)
* Математическая модель:
  - Полный конструкционный вес рамы: w_frame = Вес ригеля + Вес стоек
  - Защемляющий момент рамного узла: M_support = (q_total * L^2) / 12
  - Момент сопротивления ригеля рамы: W_beam = (b * h_beam^2) / 6
  - Совмещенное напряжение узла: sigma_comb = M_support / W_beam


3. МЕТРИКИ ОЦЕНКИ И КРИТЕРИИ БЕЗОПАСНОСТИ
-------------------------------------------------------------------------------
Расчет надежности базируется на методе предельных состояний (Сопромат).
Коэффициент запаса вычисляется относительно временного сопротивления (sigma_B):

   Коэффициент запаса (K_safe) = Предел прочности (sigma_B) / Действующее напр.

МАТРИЦА СТАТУСОВ:
* K_safe >= 1.5 -> СТАТУС: БЕЗОПАСНО (Конструкция стабильна под макс. нагрузкой)
* 1.0 <= K_safe < 1.5 -> СТАТУС: КРИТИЧЕСКИЙ УРОВЕНЬ (Пластическая деформация)
* K_safe < 1.0 -> СТАТУС: РАЗРУШЕНИЕ СИСТЕМЫ (Немедленный разлом конструкции)


4. СБОРКА И ЗАПУСК ИСПОЛНЯЕМЫХ ФАЙЛОВ JAR
-------------------------------------------------------------------------------
Проект скомпилирован на базе Java 17, JavaFX 17 и Maven.
Для исключения ошибок политик безопасности ОС при запуске JAR-архива, точка
входа перенаправлена на независимый класс-обертку (JarLauncher.java).

КОМАНДА СБОРКИ МОНОЛИТНОГО FAT JAR:
$ mvn clean package

ЗАПУСК ИСПОЛНЯЕМОГО ФАЙЛА ИЗ КОНСОЛИ:
$ java -jar target/LNL-BridgeCalc-1.0-SNAPSHOT.jar
